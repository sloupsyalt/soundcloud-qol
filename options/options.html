Keyboard shortcuts

Active on soundcloud.com when you are not typing in a field. Custom remapping is planned; defaults:

  • T Sleep timer
  • E Stop after track
  • F Focus mode
  • L Loop
  • M Mute
  • C / ⇧C Copy / copy with timestamp
  • [ ] Seek ±10s
  • - = Speed
  • ? Help
