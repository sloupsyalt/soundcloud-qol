SoundCloud QoL

Black & white controls for timer, speed, focus, loop, mute, and Discord presence.

Language

Discord

Run bridge/install-autostart.command once. Art asset key: soundcloud. Presence buttons are only visible to others.

  1. Install autostart
  2. Open Discord desktop
  3. Play a track

Features

  • Timer — presets / custom / stop after track
  • Speed — 0.75x–2x
  • Loop — repeat current track
  • Mute — toggle audio
  • Focus — quieter page
  • Copy — C track · Shift+C with timestamp
  • Recent — last tracks in the popup
